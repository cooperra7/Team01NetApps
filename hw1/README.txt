Python Packages Installed: wolframalpha  tweepy


Acquire the IP address of the raspberry pi that will be running as the server
Change the 'host' value in the client python file to reflect the server's IP
Save the changes to the client file then close it

Start the server with 'python server.py'
Run the client with 'python client.py'
To have the server answer a question, tweet a question in the form of 'netappTeam01_"<question>"' 

Answers have format '@VTNetApps Team_01 "<answer>" ###'
    The numbers are a randomly generated number from 0 to 999 to allow
    reposting reponses
